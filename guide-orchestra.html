Orchestra

Context is what a model can see right now. Orchestra is the room where you decide which agent sees what - by dragging it there.

narrated · sound on

Why this matters

Doors, not walls

Separation without sharing would be useless; sharing without visibility is how private notes leak into tools that never needed them. Orchestra is the door: context moves only when you move it, and you watch it go.

The walkthrough

The board

Open Orchestra from the sidebar. The dropdown lists every conversation from every agent - pick one and it lands on the board as a card. Add another beside it. Filter by agent with the chips along the top.

Weave

Drag a thread from one card to another and hit Weave. Cronus summarizes the source conversation and delivers the context to the target - you see exactly what moves before it lands. No copy-paste, no invisible osmosis.

When to reach for it

Any time one agent learned something another needs: the outreach thread that discovered which angle gets replies feeds the writing thread; the debugging session that found the root cause feeds the documentation job. One drag replaces re-explaining.

Clear the board

The board is a workspace, not a filing cabinet. Clear it any time - the underlying conversations are untouched.
