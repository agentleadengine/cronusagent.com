Install

Two minutes from zero to the dashboard. Your alpha invite includes access to the private code repository - the commands below work once that access is set up. You need git and Python 3.11 or newer; everything else installs itself on first run.

Works on macOS, Windows, and Linux. For the free local model: 8 GB of RAM works, 16 GB is comfortable. The installers need nothing else; installing from source needs git and Python 3.11+.

Or install from source
1

Get the code and start it

Open Terminal and run:

git clone https://github.com/agentleadengine/cronus-agent
cd cronus-agent
python3 run.py

First run creates its own private environment and installs what it needs, then serves the dashboard at http://localhost:8787/cronus.

2

Give it a free brain (recommended)

Install Ollama, then pull a small local model. Cronus finds it automatically and lists it as "local, free".

ollama pull qwen3:4b
3

Want it as a real app?

The desktop shell opens Cronus in its own window with a dock icon, and starts the backend for you. Needs Node 22+.

cd desktop
npm install
npm start

Or build the installable app: npm run dist:mac produces a DMG under desktop/dist/.

Honest note about installers: prebuilt installers are currently unsigned, so Windows SmartScreen and some antivirus tools will warn or quarantine until you allow them. That is a signing-certificate formality, not a judgment of the code. Installing from source (step 1) avoids the warnings entirely. Code signing is on the roadmap.

Questions people actually ask

What does it cost?

Cronus is currently part of Agent Lead Engine LLC's consulting package - alpha access comes with it. Once installed, running on a local model adds nothing; cloud models bill through your own accounts, on your own terms.

Is my data private?

Cronus runs on your machine, has no account system, and sends no telemetry. The only thing that ever leaves is a message you route to a cloud model, and it goes straight to that provider, nowhere else.

What do I need?

A reasonably modern computer on macOS, Windows, or Linux. 8 GB of RAM runs the free local model; 16 GB is comfortable. No accounts, no card.

Do I need all three agents?

No. Cronus works with whatever you have installed and shows an honest gray dot for the rest. Start with just the free local model and add agents when you want them.

Can I uninstall it cleanly?

Yes, and we tell you how: delete the app, then two folders in your home directory. No leftovers, no lurking services, no "are you sure" retention tricks.

How do I get an alpha code?

Email us a short intro - who you are and what you want to use it for. That is the whole application.
